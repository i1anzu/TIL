# CSS

**CSS란?** Cascading Style Sheets의 약자로, HTML로 만든 뼈대에 색, 모양, 간격, 배치 등 디자인을 입히는 언어입니다.
HTML이 "여기에 제목이 있고, 여기에 사진이 있다"라고 내용을 말한다면, CSS는 "제목은 파란색으로 하고, 사진은 둥글게 깎아줘"라고 스타일을 지시합니다.

<**CSS 기본 문법 구조**>
- **선택자(Selector)**: 스타일을 적용할 HTML 요소를 지목합니다.
- **속성(Property)**: 바꾸고 싶은 특징 (예: 색상, 크기).
- **값(Value)**: 구체적인 수치나 설정 (예: blue, 20px).
- **세미콜론(;)**: 하나의 선언이 끝났음을 알리는 기호로, 반드시 써야 합니다.

<**주요 Selector(선택자) 사용법**>
- **태그 선택자**: 	h1,	특정 태그 전체에 스타일 적용,	h1 { color: red; }
- **클래스 선택자**: .name,	HTML의 class 속성값으로 선택 (중복 가능),	.box { width: 100px; }
- **아이디 선택자**: #id,	HTML의 id 속성값으로 선택 (단 하나만),	#header { background: #eee; }

<**선택자 결합 및 우선순위**>
- **그룹 선택자**: 	여러 태그에 똑같은 스타일을 줄 때 쉼표(,)로 구분합니다.
- **자손 선택자**: 특정 태그 안에 있는 태그를 선택할 때 한 칸 띄웁니다.
- **우선순위(점)**: : 스타일이 겹칠 경우 아이디(#) > 클래스(.) > 태그 순으로 힘이 셉니다.
---
<**글꼴 (Typography) 스타일링**>
- **font-size**: 	글자 크기를 조절합니다. (단위: px, em, rem 등)
- **font-family**: 글꼴 종류를 지정합니다. (예: Arial, sans-serif)
- **font-weight**: : 글자 두께를 설정합니다. (normal, bold 또는 100~900 숫자)
- **color**: 글자 색상을 정합니다. (색상 이름, Hex 코드 #000000, RGB 등)

<**색상 (Color) 표현법**>
- **키워드**: 	red, blue, transparent(투명) 등
- **HEX 코드**: # 뒤에 6자리 16진수 조합 (예: #ff5733)
- **RGB / RGBA**: : rgb(255, 0, 0) 또는 투명도(a)를 포함한 rgba(255, 0, 0, 0.5)

<**배경 (Background) 스타일링**>
- **background-color**: 	배경색을 채웁니다.
- **font-family**: 글꼴 종류를 지정합니다. (예: Arial, sans-serif)
- **font-weight**: : 글자 두께를 설정합니다. (normal, bold 또는 100~900 숫자)
- **color**: 글자 색상을 정합니다. (색상 이름, Hex 코드 #000000, RGB 등)
