# DOM

**DOM이란?** 브라우저가 HTML 문서를 트리 구조로 시각화한 것이며, 자바스크립트와 같은 프로그래밍 언어가 HTML의 요소에 접근하고 조작할 수 있도록 연결해 주는 인터페이스(API)입니다.

> **가계도(지도)**: HTML이라는 텍스트 문서를 브라우저가 분석해서 메모리에 트리 구조로 올려둔 상태를 말합니다. (정적인 구조)

> **인터페이스**: 자바스크립트 같은 프로그래밍 언어가 그 '가계도'에 접근해서 내용을 바꾸고, 삭제하고, 추가할 수 있게 연결해 주는 통로를 말합니다. (동적인 도구)


웹페이지(HTML)는 단순한 텍스트 파일일 뿐이라서, 자바스크립트가 직접 읽거나 수정하기 어렵습니다.
그래서 브라우저는 이 텍스트를 읽어와서 나무(Tree) 모양의 객체 구조로 다시 그려내는데, 이걸 **DOM**이라고 합니다.

<**요소 선택**>
---
- **querySelector** : 조건에 맞는 첫 번째 요소를 하나만 가져옵니다. (특정 ID나 클래스 하나 찾기)
- **querySelectorAll** : 조건에 맞는 모든 요소를 한꺼번에 묶어서 가져옵니다.

<**요소 생성/수정**>
---
- **createElement** : 새로운 HTML 태그를 메모리상에 새로 만듭니다. (```<div>``` 만들기)
- **appendChild** : 새로 만든 요소를 실제 웹 화면의 특정 위치에 자식으로 집어넣습니다.
- **innerHTML** : 태그 안에 HTML 코드 통째로 내용을 넣거나 바꿉니다.
- **textContent** : 태그 안에 순수한 텍스트만 안전하게 넣거나 바꿉니다.
