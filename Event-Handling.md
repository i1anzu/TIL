# Event Handling

**이벤트 처리(Event Handling)**이란? 사용자가 웹사이트에서 하는 모든 행동(클릭, 타이핑, 스크롤 등)에 브라우저가 반응하게 만드는 기술입니다.
쉽게 말해, "어떤 사건(Event)이 일어났을 때, 어떤 동작을 할지 처리(Handling)하는 것"입니다.

<**이벤트 리스너**>
- **addEventListener** : 특정 요소(버튼, 입력창 등)에 "이런 일이 생기면 알려줘!" 라고 명령을 내리는 함수입니다.
```
버튼.addEventListener('이벤트종류', 실행할함수);
```
<**주요 이벤트**>
- **click** : 마우스로 요소를 클릭했을 때 발생합니다. (버튼 누르기 등)
- **input** : 사용자가 입력창에 글자를 칠 때마다 즉시 발생합니다. (검색어 자동완성 등)
- **change** : 입력이 끝나고 포커스가 빠지거나, 선택지(Select)가 변경되었을 때 발생합니다.
- **submit** :  양식 안의 '제출' 버튼을 누르거나, 입력창에서 Enter 키를 쳤을 때 발생합니다.

<**이벤트 객체**>
- **event** : 발생한 이벤트에 대한 모든 정보가 담긴 뭉치입니다.
- **event.target** : "그래서 누가 사고를 쳤니?"를 알려줍니다. 즉, 이벤트가 발생한 바로 그 요소를 가리킵니다.
```
여러 버튼 중 어떤 버튼을 눌렀는지
확인할 때 씁니다.
```
