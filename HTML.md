# HTML

**HTML이란?** HyperText Markup Language의 약자로, 웹페이지의 뼈대(구조)를 만드는 언어입니다.

<**HTML 기본 구조**>
- **!DOCTYPE html**: "이 문서는 최신 HTML5 표준을 따릅니다"라고 브라우저에 선언하는 문장입니다.
- **html**: 모든 HTML 요소의 최상위 부모이며, 전체를 감싸는 바구니입니다.
- **head**: 브라우저 화면에는 보이지 않지만, 제목(title), 인코딩(meta), 스타일(CSS) 등 문서의 정보를 담습니다.
- **body**: 실제 사용자의 눈에 보이는 모든 콘텐츠(글, 그림, 버튼 등)가 들어가는 곳입니다.
  
<**주요 태그**>
- **div (Division)**: 가장 많이 쓰이는 박스 모델입니다. 줄 바꿈이 일어나는 성질(Block)이 있어 레이아웃의 큰 구역을 나눌 때 사용합니다.
-  **span**: 텍스트의 특정 부분만 스타일을 줄 때 사용하는 주머니 같은 태그입니다. 줄 바꿈 없이 필요한 만큼만 공간을 차지(Inline)합니다.
- **p (Paragraph)**: 하나의 문단을 만들 때 씁니다. 위아래로 약간의 여백이 자동으로 생깁니다.
- **a (Anchor)**: 다른 페이지나 사이트로 이동하는 링크를 만듭니다. href 속성에 주소를 적어 사용합니다.
- **img (Image)**: 이미지를 화면에 불러옵니다. 종료 태그가 없는 단독 태그이며, src로 경로를 지정하고 alt로 설명글을 넣습니다.
- **form** : 로그인창이나 회원가입처럼 사용자의 데이터를 서버로 보낼 때, 입력 태그들을 감싸는 울타리 역할을 합니다.
- **input**: 사용자로부터 글자, 숫자, 비밀번호 등을 입력받는 칸입니다. type 속성에 따라 체크박스, 날짜 선택 등으로 변신합니다.
- **button**: 클릭할 수 있는 버튼입니다. 주로 form 안에서 전송 버튼으로 쓰이거나 자바스크립트 기능을 실행할 때 사용합니다.

<**시맨틱 태그**>
- **header**: 페이지의 맨 위나 특정 구역의 머리말입니다. 보통 로고, 검색창, 메뉴(Navigation)가 들어갑니다.
- **footer**: 페이지의 맨 아래에 위치합니다. 저작권 정보(Copyright), 연락처, 서비스 약관 링크 등을 넣습니다.
- **main**: 페이지의 가장 중요한 본문 내용이 들어가는 곳입니다. 한 페이지에 단 한 번만 사용하는 것이 원칙입니다.
- **section**: 주제별로 콘텐츠를 그룹화할 때 씁니다.
