# JavaScript

**JavaScript란?** 웹페이지에 생명력(동적 기능)을 불어넣기 위해 사용하는 프로그래밍 언어입니다.
웹을 구성하는 3대 요소 중 하나로, 정적인 문서를 만드는 HTML과 디자인을 입히는 CSS 사이에서 사용자의 동작에 반응하고 화면을 실시간으로 바꾸는 역할을 담당합니다.

<**변수(데이터 저장소)**>
- **let** : 값이 바뀔 수 있는 변수를 만들 때 씁니다. (점수, 게임 레벨)
- **const** : 한 번 정하면 바꿀 수 없는 상수를 만들 때 씁니다. (이름, 생일)

<**조건문(의사결정)**>
- **if/else** : "만약 ~라면 A를 하고, 아니면 B를 해라" 라는 논리를 짭니다.
```
if(비가 온다) {우산을 쓴다}
else {그냥 간다}
```
<**반복문(노가다 금지)**>
- **for/while** : 똑같은 일을 여러 번 반복할 때 씁니다.
```
1부터 100까지 숫자를 하나씩 출력하기
```
<**함수(코드 묶음/기능)**>
- 특정한 동작을 미리 정의해두고 필요할 때마다 꺼내쓰는 기능입니다.
- 화살표 함수 : 최신 문법으로 => 기호를 사용해 함수를 더 짧고 깔끔하게 만듭니다.

## this 바인딩

**일반 함수**: 함수를 어떻게 호출하느냐에 따라 this가 가리키는 대상이 계속 변합니다. (유연하지만 헷갈리기 쉬움)

**화살표 함수**: 자신만의 this를 가지지 않습니다. 대신 함수가 선언된 주변 환경(상위 스코프)의 this를 그대로 물려받아 고정됩니다.
## 생성자 함수 사용 여부
**일반 함수**: new 연산자와 함께 사용하여 새로운 객체를 만드는 생성자로 쓸 수 있습니다.

**화살표 함수**: 생성자로 사용할 수 없습니다. new를 붙여 호출하면 에러가 발생합니다.
## 문법의 간결함
**일반 함수**: function 키워드와 return 등을 명시적으로 써야 합니다.

**화살표 함수**: 내용이 한 줄이라면 { }와 return을 생략할 수 있어 훨씬 짧아집니다.

**일반 함수**는 호출 방식에 따라 this가 동적으로 변하는 반면, **화살표 함수**는 상위 스코프의 this를 그대로 유지하는 특징이 있습니다. 또한 **화살표 함수**는 생성자로 사용할 수 없으며, 문법이 간결해 콜백 함수로 쓰기에 매우 적합합니다."

<**자료구조(데이터 보관함)**>
- **배열 []** : 여러 데이터를 한 줄로 세워 저장합니다.
```
['사과', '포도']
```
- **객체 {}** : 데이터에 이름(키)를 붙여 저장합니다.
```
{이름: '홍길동', 나이: 20}
```
